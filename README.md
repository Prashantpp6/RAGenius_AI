# 📄 RAGenius AI – Multi-Document Assistant

## 🚀 Overview

RAGenius AI is a Retrieval-Augmented Generation (RAG) based application that allows users to interact with documents like resumes, PDFs, and notes.

## ✨ Features

* 📄 Resume Chatbot
* 📚 PDF Q&A
* 📝 Notes Assistant
* ❓ FAQ Bot
* 🔍 Semantic Search

## 🧠 Tech Stack

* Streamlit
* LangChain
* FAISS
* HuggingFace Embeddings
* Groq LLM

## ▶️ Run Locally

pip install -r requirements.txt
python -m streamlit run app_ui.py
